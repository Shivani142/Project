What is Bootstrap?

Bootstrap is a free collection of tools for creating websites and web application.It contains
HTML and CSS based design templates for typography,forms,buttons,navigation and other
interface components as well as optional JavaScript extentions.It is the No.1 project on
Github with 65,000+ stars and 23,600 forks (as of March 2014) (1) and has been used by
NASA and MSNBC,among many others...
